Here I showcase a project where I developed a straightforward Neural Network (NN) model to address a supervised problem. The project is organized into two main segments.
In the initial phase, the NN model aims to tackle a regression problem by approximating a basic function. The training dataset, containing 100 samples in the form of (x, y), is provided in an Excel file. Similarly, a corresponding file holds our test data, which is utilized to assess the model's performance. As is customary in Neural Networks, the selection of the number of layers and neurons within each layer is a critical hyperparameter that requires tuning based on the network's objectives. Given the simplicity of the prediction task in this phase, I explore two NN configurationsâ€”one with a single hidden layer and another with two hidden layers. The intention is to compare the performance of these two NNs and choose the one yielding more optimal results. For both training and testing the model, I employed the PyTorch library. Additionally, I utilized the Optuna library to optimize hyperparameters. The specific hyperparameters subjected to optimization include:
1) Learning Rate
2) Optimizer
3) Number of Neurons in first and second layer
Following the selection of the optimal model, I proceeded to retrain the model with and without early stopping, subsequently comparing the outcomes.
In the second phase of the project, I addressed a classification problem by implementing both a Neural Network (NN) and a Convolutional Neural Network (CNN). The dataset employed for this segment comprises 60,000 images from the Fashion MNIST dataset. While Convolutional Neural Networks (CNNs) are typically favored for image-related tasks, the simplicity of the dataset allows for the consideration of a fully connected network as well. Unlike the prior phase, the number of layers in this segment remains constant and is not treated as a hyperparameter. The hyperparameters for this segment are as follows:
1) Number of filters of the first channel
2) Number of filters of the second channel
3) Optimizer ==========> ["SGD", "RAdam", "Adam"]
4) Dropout probability (a regularization method)
5) Number of neurons in the first fully connected layer
6) Number of neurons in the second fully connected layer
7) The momentum parameter in SGD optimizer (regularization method)
8) L2 weight (regularization method)
