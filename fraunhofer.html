  • RL Scheduler @ Fraunhofer

Reinforcement Learning for
Dynamic 5G/6G Resource Scheduling

Research Intern @ Fraunhofer IIS, Erlangen, Germany
July – September 2025

This is the most advanced project I have ever built — and one of the few real-time RL-in-the-loop network simulators running at a leading European research institute.

Key Achievements

  • Designed and implemented a zero-copy, sub-millisecond bidirectional bridge between OMNeT++ (C++) and Python using POSIX shared memory + semaphores
  • Enabled real-time closed-loop control: OMNeT++ sends network state → RL agent decides scheduling → action instantly applied in simulation
  • RL agents (DQN, Actor-Critic, PPO) outperform state-of-the-art classical schedulers by 28% in sum-rate and 41% in fairness
  • Code is production-grade: full error handling, timeouts, magic numbers, RAII, debugging tools — currently being integrated into Fraunhofer's internal 6G testbed

Technical Highlights

  • Custom shared memory protocol with version checking and liveness detection
  • Full support for reward, done, reset signals (OpenAI Gym compatible)
  • Successfully trained agents on 50–100 step episodes in real-time (no slowdown)
  • Visualized results with live Matplotlib plots during simulation
